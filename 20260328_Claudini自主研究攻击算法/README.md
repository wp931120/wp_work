# Claudini 自主研究攻击算法

## 文章信息

- **创建日期**：2026-03-28

## 内容概要

本文解读了 Claudini 论文，展示了 AI 自主发现对抗性攻击算法的能力：

1. Claude Code 搭建的自主研究管道，迭代发现新的攻击算法
2. 在 GPT-OSS-Safeguard-20B 上达到 40% 攻击成功率（现有方法 ≤10%）
3. 在 Meta-SecAlign-70B 上达到 100% 攻击成功率（直接迁移）
4. 比传统 AutoML 方法（Optuna）损失值低 10 倍

## 核心观点

1. AI 自主研究在安全领域取得显著成果，不只是调参，而是发明新算法
2. 安全研究的军备竞赛正在加速，攻击发现门槛降低
3. 白盒攻击的威胁被低估，对抗训练过的模型也能被攻破
4. 研究伦理需要平衡：公开有利于社区，但也可能被恶意利用

## 内容概要

本文解读了 Google DeepMind 发布的 LIMIT 数据集引发的讨论：
1. LIMIT 数据集揭示了单向量稠密检索模型的维度天花板
2. 论文评估结果的 bug 和修正后的真相
3. 晚交互模型（ColBERT）如何突破这个限制
4. BM25、稠密模型、晚交互模型的对比分析

## 核心观点

1. 稠密模型的维度限制是数学事实，不是训练问题
2. 晚交互模型提供了突破单向量天花板的可能性
3. 评估工作比想象中更难，需要有统一可复现的基准
4. LIMIT 虽然是玩具数据集，但研究的组合检索能力在实际场景中很常见
